# 大模型高效化：阶段性证据与可复现研究框架（POC）

**本仓库旨在提供研究结论的「可复现证据」与「实验方法」，是一个研究快照，而非维护中的产品级项目。**

---

## 声明

- **定位**：证据驱动、主张即代码、报告即发布。所有门禁与结论均可由本框架内的脚本与证据复现。
- **承诺**：「这是我如何得到这些数据的方法，你可以按此复现和审查。」不承诺长期维护、Issue 响应或功能迭代。
- **用途**：求职作品集 / 技术展示。核心内容与证据已脱敏，与内部实验室叙事及未公开算法解耦。

## 四大支柱与当前成果

| 支柱 | 内容 | 本阶段最优（可对外表述） |
|------|------|---------------------------|
| **运行时显存** | 推理峰值内存门禁 | **PASS**（Peak RSS &lt;1024MB，可复现「&lt;1GB 推理」） |
| **静态压缩** | 权重压缩倍率与对齐（ratio × cosine） | 对齐 **≈0.99**，倍率 **≈4.7x**（policy 目标 20x 未达） |
| **权重层代理** | Seed × CrossBlock × ColdDust 证据链 | **cos_min≈0.9999**，证据闭环（S1/S2/S3、Field、PPL 接入） |
| **战略突破证据** | Field / 熵增益 / 可复现性 | **go=true**，determinism_delta=0，熵增益软门禁可过 |

- **门禁总判**：当前为 **NO-GO**（静态倍率、FFN 对齐等目标未达；体系与证据链已闭环）。
- **未达成**：V38 倍率 20x、FFN_SOTA 16x@cosine≥0.99 仍在攻关；本 POC 不做「已达成」宣称。

## 复现与验证

- **轻量验证**（无模型）：见 `REPRODUCTION.md` 第一节，约 30 秒可跑完核心回归。
- **全量证据**（需本地 safetensors）：见 `REPRODUCTION.md` 第二节，按顺序执行证据脚本与门禁。
- **证据含义**：`evidence/` 下为示例；Schema 见 `schemas/`。真实证据由运行脚本产出，格式与示例一致。
